# Workflow v1

1. Create/Update canon + timeline anchors
2. Outline chapter objectives
3. Draft scenes quickly
4. Run continuity pass (canon/timeline/character voice)
5. Run line-edit pass (tone, rhythm, precision)
6. Lock chapter

## Branching
- `feature/book1-outline`
- `feature/book1-draft-chXX`
- `fix/continuity-evt-###`

## Commit Convention
- `canon:` lore/rule changes
- `timeline:` event/date changes
- `character:` character sheet changes
- `draft:` prose/chapter changes
- `edit:` line edits only
